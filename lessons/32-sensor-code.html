The Sensor Decides

الزرّ يعطي خبرين اثنين: مضغوط أو مرفوع.
أمّا الحسّاس فيعطي رقمًا يتغيّر باستمرار.

اضغط ◀ لنقرأ ذلك الرقم.

قطعة تترجم الضوء إلى مقاومة Light Becomes Resistance

في الضوء تصغر مقاومته، وفي الظلام تكبر.
واللوحة لا تقيس الضوء، بل تقيس الجهد الناتج.

أطفئ ضوء الغرفة في الرسم.
الحسّاس يخبر اللوحة، واللوحة تُشعل الليد.

ينزل الضوء فينزل الرقم معه The Number Follows the Light

لا قفزة مفاجئة بين نهار وليل.
الرقم ينحدر ببطء مع غروب الشمس.

⚡

الرقم متدرّج، والليد ليس متدرّجًا: يضيء أو ينطفئ.
فمن يفصل بين الحالتين؟ أنت.

خطّ تختاره أنت بين حالتين Choosing a Threshold

📏

نختار رقمًا، وليكن ٣٠٠.
ما دونه نعدّه ظلامًا، وما فوقه نعدّه ضوءًا.
هذا الرقم اسمه threshold.

سطر يقيس وسطران يحكمان Measure, Then Judge

🌙 ➜ 💡
nightlight.ino
int x = analogRead(A0);
if (x < 300) digitalWrite(13, HIGH);
else        digitalWrite(13, LOW);

ثلاثة أسطر تصنع مصباح ممرّ حقيقيًّا.

ثلاثة أسطر تحت المجهر Under the Microscope

🔬

  1. int x = analogRead(A0)

    خذ قراءة الحسّاس واحفظها في صندوق اسمه x.

  2. if (x < 300)

    أصغر من العتبة؟ إذن المكان مظلم.

  3. else

    وإلّا فالضوء كافٍ، ولا حاجة إلى الإنارة.

لا تخمّن الرقم… قِسه Measure Before You Choose

📋

اقرأ الحسّاس نهارًا، ثم اقرأه ليلًا.
وضع العتبة بين القراءتين، لا في أوّل رقم يخطر لك.

🎓
